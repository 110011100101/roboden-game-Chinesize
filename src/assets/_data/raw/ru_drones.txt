##drone.worker : Рабочий
##drone.worker.description
Стандартный рабочий дрон.
Он может собирать ресурсы, строить и ремонтировать здания.
Эта модель может быть произведена непосредственно на колонии.
При объединении с другими рабочими и скаутами он может стать дроном 2-го тира.

##drone.scout : Скаут
##drone.scout.description
Базовый боевой дрон.
Эта модель может быть произведена непосредственно на колонии.
При объединении с другими скаутами и рабочими он может стать дроном 2-го тира.

##drone.scavenger : Стервятник
##drone.scavenger.description
Легкий боевой дрон, который может собирать металлолом в окрестностях.
Этот дрон не может выполнять другие задачи рабочих, типа строительства.
Большинство его характеристик уступают дрону боец, но и затраты на его содержание ниже.

##drone.courier : Курьер
##drone.courier.description
Дрон, который летает между колониями для улучшения их экономики.
У этого юнита есть лёгкая замедляющая пушка. Курьер немного ремонтируется после каждой успешной доставки.

##drone.cloner : Клонирователь
##drone.cloner.description
Уникальный дрон, который может клонировать любого другого дрона, кроме элитных.
Клонирование, как правило, требует меньше ресурсов, чем обычное производство.

##drone.freighter : Грузовой
##drone.freighter.description
Дрон с расширенной вместительностью.
Этот дрон может перевозить больше ресурсов за перелёт чем любая другая модель.

##drone.redminer : Алый Рудокоп
##drone.redminer.description
Единственный юнит, который может добывать специальный лавообразный ресурс.
Этот красный ресурс позволяет производить элитных дронов.

##drone.crippler : Охотник
##drone.crippler.description
Легкий боевой дрон, который атакует до трех целей на дистанции выше средней.
Затронутые цели потеряют некоторую скорость.
Этот дрон может стать невидимым для избежания фатальных повреждений.

##drone.disintegrator : Дезинтегратор
##drone.disintegrator.description
Рабочий дрон, который имеет мощное оружие - дезинтеграторную пушку.
Одиночное попадание дезинтегратора наносит огромный ущерб, но скорострельность очень низкая, а потребление энергии - высокое.

##drone.fighter : Боец
##drone.fighter.description
Надёжный боевой дрон.
Хорошая дальность стрельбы, урон и прочность.
Высокая цена содержания.

##drone.defender : Защитник
##drone.defender.description
Боевой дрон, который атакует до двух целей и заставляет их атаковать его.
По сравнению с дроном класса боец, у него меньшая огневая мощь и ниже скорость перемещения, но выше дальность атаки и защита.

##drone.prism : Призма
##drone.prism.description
Дорогой, но мощный боевой дрон, атаки которого становятся еще более сильными при объединении с другими дронами того же типа.
Сила в количестве!

##drone.servo : Ускоритель
##drone.servo.description
Дрон поддержки, который улучшает подвижность колонии.
Колония будет перемещаться на большие расстояния с более высокой скоростью.
Этот эффект складывается от нескольких дронов этого типа.

##drone.repeller : Шокер
##drone.repeller.description
Рабочий дрон, оснащенный легкой ЭМП-пушкой, которая может атаковать до двух целей.
Его атаки могут лишить противника возможности атаковать.

##drone.repair : Ремонтник
##drone.repair.description
Дрон, который может починить поврежденных союзных дронов.
Он не может починить сам себя, поэтому рекомендуется иметь несколько ремонтных роботов.
Ремонтники имеют очень высокую цену содержания.

##drone.recharger : Заряжатель
##drone.recharger.description
Юнит, который восстанавливает энергию у союзных дронов.
Это позволяет колонии работать более эффективно.

##drone.generator : Генератор
##drone.generator.description
Рабочий дрон, который снижает затраты на содержание юнитов колонии.
Большее количество дронов этого типа усиливают этот эффект.

##drone.mortar : Мортира
##drone.mortar.description
Дрон с очень дальнобойной атакой против наземных целей.
Его атаки достаточно точные, чтобы поражать стационарные и движущиеся цели издалека.
У этого дрона очень высокая прочность.

##drone.antiair : Ракетчик
##drone.antiair.description
Дрон с ракетной установкой против воздушных целей в качестве оружия.
Он очень эффективен, но также довольно хрупок и не может атаковать наземные цели.

##drone.trucker : Дальнобойщик
##drone.trucker.description
Улучшенная версия курьера.
У этой модели выше грузоподъёмность, атака и защита.

##drone.stormbringer : Громовержец
##drone.stormbringer.description
Дрон-рабочий, который имеет тяжелую штормовую пушку, наносящую ЭМП-урон двум целям.
Его атаки могут лишить противника возможности атаковать.
Громовержец сохраняет свойства дрона класса генератор.

##drone.destroyer : Разрушитель
##drone.destroyer.description
Один из наиболее универсальных и мощных боевых дронов.
Высокая атака и защита за высокую стоимость содержания.
Эти дроны могут стать авангардом войск колонии.

##drone.marauder : Марадёр
##drone.marauder.description
Улучшенный сборщик, который получает возможность маскировки.
Он атакует до трех целей и причиняет замедляющий эффект.

##drone.guardian : Хранитель
##drone.guardian.description
Улучшение дрона-защитника, который может производить саморемонт.
У этой модели атака ниже, чем у налётчика, однако выживаемость значительно выше.

##turret.gunpoint : Бункер
##turret.gunpoint.description
Прочная защита, которая быстра и дешева в постройке.
Пара таких сооружений может остановить небольшую армию вражеских наземных юнитов.
Бункер уязвим к атакам вражеских летающих юнитов.

##turret.beamtower : Лучевая Башня
##turret.beamtower.description
Платформа с высокоэффективным дальнобойным энергетическим оружием.
У неё нет прочности бункера, и построить её сложнее.
Скорость стрельбы снижается при непрерывном огне и восстанавливается вне боя.
Башня уязвима к атакам вражеских наземных юнитов.

##turret.tetherbeacon : Навигационный Маяк
##turret.tetherbeacon.description
Вспомогательное сооружение, которое не атакует. Маяк ускоряет ближайших юнитов и колонии.
Это строение является альтернативой дрону класса ускоритель.

##drone.kamikaze : Священный Ветер
##drone.kamikaze.description
Лёгкий и недорогой рабочий дрон.
При высоком повреждении может влететь в противника, взрывая себя ради приченения массивного урона.
Рецепт создания позволяет производить этих дронов в больших количествах.

##drone.skirmisher : Налётчик
##drone.skirmisher.description
Высокотехнологичный боевой юнит с хорошим залповым уроном на средних дистанциях.
Оружие налётчика наносит только половину урона по летающим целям.
Этот дрон может медленно ремонтировать себя находясь вне боя.
