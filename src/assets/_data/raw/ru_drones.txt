##turret.harvester : Собиратель
##turret.harvester.description
Бронированный комбайн, который ездит и собирает ресурсы для колонии.
Собиратель добывает только минералы, а не нефть или обломки.
У собирателя нет никакого оружия.

##turret.siege : Огненная Буря
##turret.siege.description
Осадная ракетная установка.
Может атаковать только крупные наземные цели.
После нескольких залпов ей требуется много времени на перезарядку.

##turret.refinery : Очиститель
##turret.refinery.description
Вспомогательное сооружение, которое не атакует. Очиститель служит улучшенной точкой приёма ресурсов.
Помимо увеличения доходов, он хранит часть излишков в себе.
Эти дополнительные ресурсы распределяются между колониями по мере надобности.

##turret.sentinelpoint : Смерч
##turret.sentinelpoint.description
Лазерная турель залпового огня, которая атакует группы наземных и воздушных целей.
Эта установка требует постоянную поддержку трёх рабочих дронов для функционирования.
При выведении из строя любого из дронов обслуживания, турель отключается.

##turret.gunpoint : Бункер
##turret.gunpoint.description
Прочная защита, которая быстра и дешева в постройке.
Пара таких сооружений может остановить небольшую армию вражеских наземных юнитов.
Бункер уязвим к атакам вражеских летающих юнитов.

##turret.beamtower : Лучевая Башня
##turret.beamtower.description
Платформа с высокоэффективным дальнобойным энергетическим оружием.
У неё нет прочности бункера, и построить её сложнее.
Скорость стрельбы снижается при непрерывном огне и восстанавливается вне боя.
Башня уязвима к атакам вражеских наземных юнитов.

##turret.tetherbeacon : Навигационный Маяк
##turret.tetherbeacon.description
Вспомогательное сооружение, которое не атакует. Маяк ускоряет ближайших юнитов и колонии.
Это строение является альтернативой дрону класса ускоритель.
